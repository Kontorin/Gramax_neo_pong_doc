## 1\. Назначение программы

NEO ping-pong -- учебная двумерная игра на Python (Pygame), представляющая модифицированную версию классического пинг-понга. Программа реализует игровой процесс двух игроков, управляющих ракетками для отбивания светящегося мяча. Уникальной особенностью является система взрывчатых мин, появляющихся на поле после каждого соприкосновения ракетки с мячом. При контакте мины с мячом происходит взрыв, мяч ускоряется и отскакивает в обратном направлении. Цель игры -- предотвратить попадание мяча за границы поля на своей стороне и набрать большее количество очков противника.

## 2\. Условия выполнения программы

**Требования к аппаратуре:**

-  Персональный компьютер с операционной системой Windows

-  Процессор с минимальной частотой 1 ГГц

-  Оперативная память: не менее 512 МБ

-  Видеокарта, поддерживающая разрешение 1268×850 пикселей

**Требования к программному обеспечению:**

-  Python версии 3.8 и выше

-  Библиотека Pygame версии 2.0 и выше

## 3\. Выполнение программы

**Установка и запуск при использовании Git:**

Клонируйте репозиторий:

```bash
git clone https://github.com/Kontorin/neo_ping-pong.git
```

Перейдите в папку проекта:

```bash
cd neo_ping-pong
```

Активируйте виртуальное окружение:

```bash
source venv/Scripts/activate
```

Запустите игру:

```bash
python main.py
```

**Управление игроком 1 (левая ракетка, красная):**

-  **W** -- движение ракетки вверх

-  **S** -- движение ракетки вниз

**Управление игроком 2 (правая ракетка, синяя):**

-  **↑** (стрелка вверх) -- движение ракетки вверх

-  **↓** (стрелка вниз) -- движение ракетки вниз

**Завершение программы:**

-  Нажмите кнопку закрытия окна (X) на строке заголовка программы

## 4\. Сообщения оператору

**Ошибка: "ModuleNotFoundError: No module named 'pygame'"**

-  *Решение:* Установите библиотеку Pygame командой `pip install pygame`

**Ошибка: "FileNotFoundError: assets/..."**

-  *Решение:* Убедитесь, что папка `assets` с графическими ресурсами находится в корневой папке проекта

**Окно игры не открывается**

-  *Решение:* Проверьте версию Python (требуется 3.7+). Запустите `python --version`

**Низкая частота кадров или притормаживание**

-  *Решение:* Закройте другие приложения; проверьте характеристики видеокарты. В файле `settings.py` можно понизить значение `FPS` с 120 на 60

**Игрок не может управлять ракеткой**

-  *Решение:* Убедитесь, что окно игры находится в фокусе (активно). Переклюитесь в окно игры клавишей Tab или щелчком мыши

**Мяч прошел сквозь ракетку**

-  *Это нормально при высокой скорости;* целевое поведение при развитии мяча через мины

**Мины не появляются**

-  *Решение:* Проверьте файлы ракеток в папке `assets`. Мины должны спавниться после каждого удара по мячу
